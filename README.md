### Instalación

Para comenzar, clona el repositorio e introduce los siguientes comandos en tu terminal

 ```
 mvn clean install 
 mvn spring-boot:run
```

> La aplicación iniciará en http://localhost:8082.

### Dependencias externas

* <p style="font-size:11px;">PostgreSQL
